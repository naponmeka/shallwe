# shallwe

## Usage

```
NAME:
   shallwe - Shall we build?

USAGE:
   shallwe [global options] command [command options] [arguments...]

VERSION:
   0.0.1

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --base value     the commit in which you want to compare (default: "HEAD~1")
   --folders value  the folders/files in which you want to watch (seperate with ',')
   --help, -h       show help
   --version, -v    print the version
```
